Echo development no longer supports the use of Internet explorer

Story on : 2/10/2022

As of February 10th 2022 Echo development no longer supports the use of Internet explorer and will discontinue support for use of Internet explorer.

Microsoft edge will end its support and and will discontinue for windows in June 25 of 2022. We will no longer develop sites to be used for Internet explorer.

Why are we doing this? For an internet service that is no longer providing updates and has and will end the service we can no longer provide support for sites. Our websites are not built to support Internet explorer and should not be used on Internet explorer.

Internet Explorer mode in Microsoft Edge enables backward compatibility and will be supported through at least 2029. Additionally, Microsoft will provide notice one year prior to the retirement of IE mode. The alternative for most businesses will be Microsoft Edge with IE mode. Microsoft created its IE mode for Edge a couple of years ago, and it has allowed businesses to adopt the new Chromium-based browser for older legacy websites. IE mode supports older ActiveX controls and legacy sites, which are surprisingly still used by many businesses. Microsoft is promising to support this IE mode in Edge until through at least 2029.

We recommend using Google, Google Chrome, Microsoft Edge, or any other Internet service
